ESP32 Composite Video: The Ultimate Setup Guide

By Match | 2 min read |
June 16, 2026

 

This comprehensive guide covers everything you need to generate composite video with an ESP32, including the wiring diagram, source code, and a full parts list

Hardware & Components Required

NameDescriptionAmount
ESP32ESP32 Development Board (ESP32-WROOM-32)x1
RCA CableFor Video Transferx1
Resistor (230Ω - 470Ω)For GPIO safe usage (current limiting and voltage matching)x1
Bread BoardFor building and testing the circuit without soldering (temporary connections).x1
Jumper WiresFor Connectionsx3
ComputerFor Build and Flashing projectx1
USB CableUSB-A to USB-C or Micro-USBx1
⚠️ Important Note: This tutorial is specifically designed for the classic ESP32 (ESP32-WROOM-32). Other series such as ESP32-S2, S3, C3, C6, H2, and P4 are NOT supported by the composite video library and will not work with this guide.

Software & Tools Required

Before connecting the hardware, you need to set up your computer with the required software.

Install Arduino IDE or Platformio

Install Arduino IDE Or Platformio And Install ESP32 Boards.

Install Library

in Arduino:

  1. In the Arduino IDE, go to Sketch > Include Library > Manage Libraries.

  2. Search for ESP_8_BIT Color Composite Video Library
  3. Click Install.
in Platformio:
  1. Click Platformio Icon From Left Side Bar
  2. From Quick Access click Libraries 
  3. Search for ESP_8_BIT Color Composite Video Library and click it
  4. click Add to Project 
  5. Select your project and click Add button

Set Esp32 Board

  1. From top, Select

Flash The Example Code

Copy And Paste This Code.
#include <ESP_8_BIT_GFX.h>

ESP_8_BIT_GFX video(true /* true = NTSC | false = PAL */, 8);

void setup() {
  video.begin();
}

void loop() {
  video.waitForFrame(); // Wait For Next Frame
  video.fillScreen(0); // Fill Screen With Color Black
  video.setTextColor(0xE4);
  video.setCursor(10,10);
  video.println("Esp32 Composite Video OUT");
  video.setTextColor(0x43);
  video.setCursor(10,10);
  video.println("Example With 8-bit Colors");
  video.setTextColor(0x5C);
  video.setCursor(10,10);
  video.println("And Created By MatchStudios");
}
